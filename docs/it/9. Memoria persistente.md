# 9. Memoria persistente

La memoria `M` misura quanto il sistema conserva tracce operative del proprio passato. Non è sufficiente che un sistema abbia uno storage. La memoria rilevante per la coscienza deve influenzare la dinamica futura del sistema.

Una definizione operativa minimale può essere:

```math
M_\tau(S) = I(S_t; S_{t-\tau})
```

Tuttavia, questa misura grezza può essere ingannevole. Una pietra conserva tracce fisiche del passato, ma questo non implica memoria funzionale nel senso cognitivo. Serve quindi una versione condizionata rispetto al comportamento futuro:

```math
M^*_\tau(S) = I(S_{t-\tau}; S_{t+1} \mid I_t)
```

Questa formula misura quanto lo stato passato contribuisce allo stato futuro, oltre l'input immediato. In termini concettuali, un sistema ha memoria persistente rilevante se il suo passato modifica il modo in cui elabora il presente.

La tesi è:

> senza memoria persistente non esiste continuità del soggetto, e senza continuità del soggetto non esiste coscienza in senso forte.

Questa affermazione è falsificabile in linea di principio: se si dimostrasse l'esistenza di esperienza soggettiva in un sistema privo di qualunque dipendenza causale dal proprio passato, la teoria verrebbe indebolita o falsificata.
