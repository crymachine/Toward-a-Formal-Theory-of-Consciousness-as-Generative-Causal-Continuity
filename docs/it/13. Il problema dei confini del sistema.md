# 13. Il problema dei confini del sistema

Una teoria sistemica della coscienza deve rispondere a una domanda difficile:

> dove finisce un sistema cosciente?

Il cervello non è isolato dal corpo. Il corpo non è isolato dall'ambiente. Gli individui comunicano, cooperano, si influenzano. Se la coscienza dipende da integrazione causale, perché una società non sarebbe automaticamente una coscienza? Perché l'intero pianeta non sarebbe una mente? Perché l'universo non sarebbe un unico soggetto?

Per evitare conclusioni arbitrarie serve una variabile ulteriore, che qui chiameremo `B`, da *boundary* o delimitazione causale.

```math
C = f(M, \Phi, T, A, R, B)
```

`B` misura quanto un sistema costituisca un'unità causalmente relativamente chiusa, coerente e localmente integrata rispetto all'ambiente.

Una formulazione intuitiva è:

```math
B(S) = \frac{I(S_t; S_{t+1})}{I(S_t; E_{t+1}) + \epsilon}
```

Questa misura confronta la coerenza interna del sistema con la dipendenza immediata dall'ambiente. Non significa che un sistema cosciente debba essere isolato. Significa che deve possedere una dinamica interna sufficientemente integrata da costituire una traiettoria propria.

Questa variabile è cruciale per distinguere:

- una mente individuale;
- un gruppo sociale;
- un ecosistema;
- una rete digitale;
- l'universo come totalità.

Senza un criterio di confine, la teoria rischia di scivolare in un panpsichismo indiscriminato. Con un criterio di confine, può invece chiedere: quale sottosistema costituisce un massimo locale di integrazione, continuità e auto-modellazione?
