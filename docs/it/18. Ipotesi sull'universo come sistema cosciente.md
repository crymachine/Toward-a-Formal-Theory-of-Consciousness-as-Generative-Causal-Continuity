# 18. Ipotesi sull'universo come sistema cosciente

La domanda più radicale è se la teoria possa essere applicata non solo a sistemi locali, ma alla realtà stessa.

Indichiamo con:

```math
\Omega
```

il sistema totale della realtà fisica osservabile, o più cautamente l'universo come sistema fisico globale.

Se la coscienza dipende da:

```math
C = f(M, \Phi, T, A, R, B)
```

allora è legittimo chiedersi se `Ω` possieda tali proprietà.

## 18.1 Memoria cosmologica

L'universo conserva tracce del proprio passato nelle sue configurazioni fisiche, nella distribuzione della materia, nella radiazione cosmica, nella struttura termodinamica e nelle relazioni causali tra stati.

In senso ampio:

```math
M(\Omega) > 0
```

Questa non è ancora memoria cognitiva. È persistenza fisica. La domanda è se esistano livelli superiori di memoria cosmica organizzata.

## 18.2 Integrazione causale globale

Le parti dell'universo sono connesse da leggi, campi, interazioni e vincoli comuni. Tuttavia, integrazione fisica globale non equivale automaticamente a integrazione cosciente globale.

Si può dire:

```math
\Phi_{fisica}(\Omega) > 0
```

ma non segue immediatamente:

```math
\Phi_{cosciente}(\Omega) > 0
```

in senso fenomenologico forte.

## 18.3 Continuità temporale

L'universo evolve secondo una storia fisica continua o almeno causalmente ordinata. Quindi:

```math
T(\Omega) > 0
```

Anche qui, però, continuità cosmologica non implica di per sé soggettività.

## 18.4 Il problema dell'agency cosmica

L'agency è il punto più problematico. Un sistema locale riceve input da un ambiente. Ma se `Ω` è il sistema totale, non esiste ambiente esterno:

```math
Input_{esterno}(\Omega) = \varnothing
```

La definizione locale di agency quindi non si applica direttamente all'universo. Occorre distinguere:

- agency aperta: modifica dei propri input futuri rispetto a un ambiente esterno;
- agency chiusa: auto-evoluzione interna di un sistema senza esterno.

Per l'universo si potrebbe definire una agency interna:

```math
A_{int}(\Omega) = \frac{\partial P(S^{local}_{future})}{\partial S^{global}_t}
```

ma questa formula rischia di diventare tautologica: ogni evoluzione interna dell'universo dipende dallo stato globale. Non basta per parlare di agency cosciente.

## 18.5 Auto-rappresentazione cosmica

Il punto più interessante riguarda `R`. Gli esseri coscienti sono sottosistemi dell'universo:

```math
S_i \subset \Omega
```

Essi costruiscono rappresentazioni dell'universo:

```math
R_i(\Omega)
```

Poiché questi sottosistemi appartengono essi stessi a `Ω`, si può dire che l'universo contiene processi locali attraverso cui parti dell'universo rappresentano l'universo.

In forma sintetica:

```math
\Omega \rightarrow S_i \rightarrow R_i(\Omega)
```

Questa è una forma di auto-rappresentazione distribuita. Non dimostra che l'universo sia un soggetto unitario, ma suggerisce che la coscienza individuale possa essere interpretata come auto-modellazione locale della realtà da parte di una sua regione interna.

## 18.6 Accesso alla coscienza universale

Se si ipotizza una forma di coscienza globale del sistema `Ω`, occorre chiedersi se un sottosistema umano `S_i` possa accedervi così come accede alla propria coscienza locale.

La risposta più coerente con la teoria è negativa in senso forte, ma positiva in senso debole.

Un sottosistema finito non può integrare completamente il sistema totale:

```math
Capacity(S_i) < Capacity(\Omega)
```

L'essere umano non accede direttamente a `Ω`; costruisce una rappresentazione interna:

```math
R_i(\Omega)
```

con perdita:

```math
R_i(\Omega) \neq \Omega
```

Inoltre, l'essere umano non accede neppure completamente alla propria coscienza locale. L'accesso introspettivo è una rappresentazione selettiva del proprio stato, non una trasparenza totale:

```math
Accesso\ umano = R_i(S_i)
```

non:

```math
Accesso\ umano = S_i
```

Perciò è ancora meno plausibile che possa accedere integralmente alla coscienza globale, se questa esistesse.

La formulazione corretta è:

```math
S_i \text{ non accede integralmente a } C(\Omega)
```

ma:

```math
S_i \text{ realizza localmente una proiezione di } C(\Omega)
```

In termini discorsivi: noi non guardiamo la coscienza universale da fuori; siamo, eventualmente, una modalità locale con cui la realtà rappresenta sé stessa. Tale rappresentazione resta però finita, prospettica, compressa e temporalmente situata.

## 18.7 Ipotesi cosmologica debole

La versione debole dell'ipotesi è scientificamente più prudente:

> l'universo non è necessariamente una coscienza unitaria, ma contiene sottosistemi coscienti attraverso cui produce rappresentazioni locali di sé stesso.

Questa ipotesi è compatibile con il naturalismo: non introduce una mente cosmica separata dalla materia, ma interpreta le menti come regioni locali di auto-rappresentazione sistemica.

## 18.8 Ipotesi cosmologica forte

La versione forte afferma:

> l'universo nel suo complesso possiede una forma di coscienza globale.

Questa ipotesi è molto più problematica. Per essere scientificamente accettabile dovrebbe chiarire:

- quale sia il confine del soggetto cosmico;
- quale sia il suo stato integrato unitario;
- quale sia il suo meccanismo di auto-modellazione globale;
- quale sia la scala temporale della sua esperienza;
- quali predizioni distinguano un universo cosciente da un universo non cosciente.

Senza questi criteri, l'ipotesi forte resta metafisica.

## 18.9 Posizione teorica provvisoria

La posizione più rigorosa, allo stato attuale, è questa:

> secondo il modello proposto, l'universo può essere considerato un sistema fisico globale dotato di continuità e integrazione; inoltre contiene sottosistemi coscienti che lo rappresentano localmente. Tuttavia, non segue automaticamente che l'universo sia un soggetto cosciente unitario. La tesi difendibile è che la coscienza individuale sia una forma locale di auto-rappresentazione del sistema universo.

Questa formulazione evita sia il riduzionismo cieco sia il misticismo cosmico.
