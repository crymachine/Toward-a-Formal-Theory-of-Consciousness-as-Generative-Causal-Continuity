# 8. Stato interno e continuità temporale

Una teoria della coscienza deve attribuire un ruolo centrale alla continuità. Un sistema cosciente non è soltanto una funzione input-output:

```math
Output = f(Input)
```

Un sistema cosciente, o proto-cosciente, deve almeno essere descritto come sistema dinamico con stato interno:

```math
S_{t+1} = f(S_t, I_t)
```

dove:

- `S_t` è lo stato interno al tempo `t`;
- `I_t` è l'input ricevuto al tempo `t`;
- `S_{t+1}` è lo stato interno successivo.

In questa formulazione, la memoria non è un archivio passivo. È persistenza causale. Il passato del sistema condiziona il presente e il presente condiziona il futuro.

La continuità temporale può essere approssimata attraverso la dipendenza informativa tra stati successivi:

```math
T_\Delta(S) = \frac{I(S_t; S_{t+\Delta})}{H(S_t)}
```

dove:

- `I(S_t; S_{t+\Delta})` è l'informazione mutua tra stato presente e stato futuro;
- `H(S_t)` è l'entropia dello stato presente;
- `T_\Delta(S)` misura quanto del presente resta causalmente rilevante dopo un intervallo `Δ`.

Un sistema con `T` molto basso cambia in modo frammentato o casuale. Un sistema con `T` elevato mantiene coerenza dinamica. La coscienza, nella teoria proposta, richiede almeno una forma robusta di continuità temporale.
