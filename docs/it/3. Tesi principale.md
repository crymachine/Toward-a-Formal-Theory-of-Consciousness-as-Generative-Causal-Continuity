# 3. Tesi principale

La tesi proposta può essere formulata così:

> La coscienza è il grado con cui un sistema dinamico delimitabile conserva una continuità interna integrata nel tempo, usa tale continuità per modellare sé stesso e l'ambiente, e modifica attivamente la distribuzione dei propri input futuri.

Questa tesi contiene cinque affermazioni fondamentali.

La prima affermazione è che la coscienza non è binaria. Non esiste necessariamente un confine netto tra sistema cosciente e sistema non cosciente. Esiste piuttosto un gradiente:

```math
0 \leq C \leq 1
```

oppure, in forma più generale:

```math
C \in \mathbb{R}^{+}
```

La seconda affermazione è che la coscienza richiede persistenza temporale. Un sistema che producesse soltanto output istantanei in risposta a input istantanei, senza conservare tracce causali del proprio passato, non avrebbe continuità esperienziale.

La terza affermazione è che la coscienza richiede integrazione. Non basta che un sistema abbia molte parti o molta informazione: le parti devono contribuire a una dinamica unitaria, non riducibile alla somma di processi isolati.

La quarta affermazione è che la coscienza cresce con l'agency. Un sistema più cosciente non riceve semplicemente input: seleziona, induce, cerca, modifica e struttura i propri input futuri in funzione del proprio stato interno.

La quinta affermazione è che la coscienza implica una forma di auto-modellazione. Non necessariamente nel senso umano pieno dell'introspezione linguistica, ma almeno nel senso minimo di una rappresentazione operativa del proprio stato, dei propri limiti e della propria relazione con l'ambiente.
