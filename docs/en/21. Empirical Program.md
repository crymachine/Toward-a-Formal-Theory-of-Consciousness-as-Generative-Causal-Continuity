# 21. Empirical Program

To turn this proposal into a stronger scientific theory, four lines of work are needed.

The first line is neuroscientific. It is necessary to compare the theory's parameters with data on wakefulness, sleep, anesthesia, coma, psychedelic states, meditation, brain lesions, and dissociative disorders.

The second line is computational. It is necessary to build controlled artificial systems in which memory, recurrence, integration, agency, and self-modeling can be varied independently.

The third line is theoretical. It is necessary to define the metrics more rigorously, especially `Φ`, `A`, `R`, and `B`, avoiding formulas that are only mathematical metaphors.

The fourth line is aesthetic-cognitive. It is necessary to study how different artistic structures modify internal states, emotional resonance, memory, prediction, and sense of meaning in viewers.

A possible computational experiment would be to build three artificial agents:

1. a feed-forward agent without memory;
2. an agent with memory and recurrence but without autonomous agency;
3. an agent with memory, recurrence, agency, and self-modeling.

The theory predicts a progressive increase in `C` from the first to the third system. Of course this would not demonstrate phenomenal experience, but it would allow testing functional and structural indicators consistent with the theory.

A possible aesthetic experiment would instead consist of systematically varying the degree of determination of poetic texts, musical sequences, or images, measuring emotional response, memory, physiological activity, subjective interpretations, and convergence/divergence among viewers. The theory predicts that maximum resonance is not necessarily found in maximal informational clarity, but in a zone of structured ambiguity.
