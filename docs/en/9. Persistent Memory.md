# 9. Persistent Memory

The memory `M` measures how much the system preserves operative traces of its own past. It is not enough for a system to have storage. The memory relevant for consciousness must influence the system's future dynamics.

A minimal operational definition can be:

```math
M_\tau(S) = I(S_t; S_{t-\tau})
```

However, this rough measure can be misleading. A stone preserves physical traces of the past, but that does not imply functional memory in the cognitive sense. Therefore a conditioned version with respect to future behavior is needed:

```math
M^*_\tau(S) = I(S_{t-\tau}; S_{t+1} \mid I_t)
```

This formula measures how much the past state contributes to the future state, beyond the immediate input. In conceptual terms, a system has persistent memory if its past modifies how it processes the present.

The thesis is:

> without persistent memory there is no continuity of the subject, and without subject continuity there is no strong consciousness.

This claim is falsifiable in principle: if it were demonstrated that subjective experience exists in a system devoid of any causal dependence on its past, the theory would be weakened or falsified.
